# FantaChat Beta Feedback Log

Stato: template operativo.

Usare una riga per ogni problema o feedback rilevante.

## Legenda severita

- P0: blocca la beta.
- P1: grave, blocca allargamento.
- P2: fastidioso, aggirabile.
- P3: miglioramento o richiesta futura.
